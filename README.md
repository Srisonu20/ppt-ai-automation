# ppt-ai-automation
Introduction through PPT_AI website and automating slides by scrolling step by step.

- Web Automation

## Features
- Automated interaction with PPT AI website
- Step-by-step slide navigation (scrolling/sliding)
- Reduced manual effort and improved productivity

## Workflow
1. Open PPT AI website using Selenium
2. Enter to created ppt
3. Automatically navigate through slides one by one
